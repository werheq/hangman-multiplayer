# Multiplayer Hangman Game

A simple multiplayer Hangman game supporting 1v1, 2v2, 3v3, and 4v4 matches with real-time chat. Just enter your username and play - no registration required!

## Features

- **Simple Login**: Just enter any username (like Kahoot!)
- **Username Check**: If a username is already in use by an online player, you'll need to pick another
- **Game Modes**: 1v1, 2v2, 3v3, 4v4
- **Room System**: Create/join rooms with optional passwords
- **Real-time Chat**: Chat with other players in rooms

## How to Play

1. **Enter your username** and click "Join Game"
   - If the username is taken, try another one
   - Usernames are only reserved while you're online
2. **Create a Room** or join an existing one
3. **Wait** for players to join
4. **Host starts** the game
5. **Take turns** guessing letters
6. **Chat** with other players

## Technical Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js, Socket.io
- **Data**: In-memory storage (no database needed)
